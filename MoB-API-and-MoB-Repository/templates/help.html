- My segmentation has the highest score, Why it doesn't show up on the GT tab of the Web Page I segmented?

The segmentations created by the users compose a pool of segmentations called "Ground Truth", the best segmentation is created automatically from that pool, taking all the shared characteristics between all the segmentations. This is the segmentation that is showed on the GT tab.
